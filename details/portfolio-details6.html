Hotel Management System

Project Details

  • Category: App Development
  • Client: Java
  • Project date: 18 March, 2021

A Hotel Management System Taking in the customer's details and the room number they want to stay. If the customers surpass the number 8 they would be kept in waiting list, This is done using java Queue. You can find the customer's room number, You can delete the customer from a room, You can store the customer Data in Array and also the program has some other Useful features.
