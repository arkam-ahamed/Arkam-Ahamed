Quiz

Project Details

  • Category: Front-end development
  • Client: HTML|CSS|JavaScript
  • Project date: 05 May, 2021

A Quiz App Which renders 3 Questions with a time interval of 10seconds. If the user marks the correct answer he would be scoring points and making the light go green and red if wrong. At the end the results is shown with a face emoji according to the results taken.
